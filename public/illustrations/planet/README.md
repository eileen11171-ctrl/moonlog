# planet 素材

七階段成長合成圖放這裡，檔名固定：

planet-stage-{n}-{slug}@{1x|2x}.webp
slug: moonlit / rosebud / rosebloom / cat / lighthouse / beacon / boathome

目前已放入：
- planet-stage-0-moonlit@1x.webp / @2x.webp ← 由你定稿的母版產生

其餘階段（1–6）生成後依此命名丟進來，App 會自動讀 manifest 顯示。
未放入的階段，UI 會顯示「紙感佔位框」，不會壞掉。

備註：母版目前為奶白紙底（與卡片紙色一致，星球自然浮著）。
若日後想要真正透明背景，去背後以同檔名覆蓋即可。
